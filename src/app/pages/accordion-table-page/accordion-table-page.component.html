Il componente Accordion table serve per avere pronto all'uso una tabella espandibile in Angular. Questa tabella consente di visualizzare in dettaglio le informazioni di ciascuna riga facendo clic su di esse.

Caratteristiche Principali
  • Espansione delle Righe: le righe della tabella possono essere espandibili per visualizzare ulteriori dettagli.
  • Espansione delle Righe: è possibile personalizzare le intestazioni delle colonne e i dettagli visualizzati nelle righe espandibili.
  • Espansione delle Righe: la tabella supporta la paginazione per la gestione di grandi set di dati.
  • Espansione delle Righe: è possibile ordinare le righe della tabella in base a specifiche colonne

Proprietà
Nome Descrizione
@Input() data: any[] Array contenente i dati da visualizzare nella tabella.
@Input() dataDetails: any[] Array contenente i dettagli aggiuntivi associati ai dati principali.
@Input() headers: Header[] Array di oggetti Header, che contiene gli attributi e le intestazioni di colonna da visualizzare nella tabella.
@Input() extras: Header[] Array di oggetti Header, che contiene informazioni aggiuntive da visualizzare nella tabella.
@Input() templateRef: TemplateRef<any> Riferimento al template utilizzato per costruire la visualizzazione grafica dei dettagli della tabella.
@Input() singleOpen: boolean Booleano che specifica se è consentito aprire solo un dettaglio alla volta.
@Input() isSortable: boolean Booleano che specifica se è consentito ordinare la tabella cliccando sull'intestazione della colonna.
@Input() maxSize: number Numero massimo di pagine da visualizzare nella paginazione.
@Input() pageSize: number Numero di elementi per pagina nella tabella.
@Input() collectionSize: any Numero totale di elementi nella raccolta di dati.
@Input() page: number Numero di pagina corrente.
@Input() disabled: boolean Booleano che specifica se la paginazione è disabilitata.
@Input() boundaryLinks: boolean Booleano che specifica se mostrare i link "Prima" e "Ultima" nella paginazione.
@Input() directionLinks: boolean Booleano che specifica se mostrare i link "Successiva" e "Precedente" nella paginazione.
@Output() detailsEmitter: EventEmitter<any> EventEmitter per catturare le informazioni della riga cliccata.
@Output() sortableEmitter: EventEmitter<SortEvent> EventEmitter per catturare le informazioni sull'intestazione della colonna cliccata e sull'ordine da applicare.
@Output() changePageEmitter: EventEmitter<number> EventEmitter per catturare il numero di pagina selezionato.
@Output() selectEmitter: EventEmitter<number> EventEmitter per catturare la selezione di un elemento.
@Output() detailsUpdate: EventEmitter<any> EventEmitter per aggiornare i dettagli.
